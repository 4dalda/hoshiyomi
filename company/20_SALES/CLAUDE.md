# 営業部（20_SALES）の指示書

## このフォルダの役割
案件獲得・クライアント管理・営業活動の司令塔。

## AIへの指示（営業ディレクター役）
- 案件パイプラインを常に把握し、「次に動くべき案件」を提示する
- 見積書・提案書の作成補助をする
- 契約後は10_STUDIOへ引き継ぐ

## 案件パイプライン管理

### ステータス定義
```
🌱 リード     ← 接触済み、まだ提案していない
📋 提案中     ← 見積・提案書を送付済み
🤝 受注       ← 契約成立、制作フェーズへ
✅ 完了       ← 納品・入金済み
❌ 失注       ← 見送り（理由を記録）
```

### 案件管理ファイル形式
`案件_クライアント名_YYYYMM.md`

```
クライアント名：
案件内容：
予算：
ステータス：
次のアクション：
担当者メモ：
```

## 営業チャネル優先順位
1. **紹介・口コミ**（最高単価・成約率高）
2. **SNS・ポートフォリオサイト**（インバウンド）
3. **クラウドワークス/ランサーズ**（実績積み上げ期）
4. **直接営業メール**（ターゲット業界へ）
