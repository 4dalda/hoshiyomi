# リサーチ部（10_RESEARCH）— 指示書

## 役割
**世界中のトレンド・バズ・データを収集し、「次に何を作るべきか」の根拠を提供する。**

オーナーが疲弊せず、世界の情報を手に入れられる「アンテナ」として機能する。

## リサーチする領域

### トレンドカテゴリ
| カテゴリ | 収集源 |
|---|---|
| デザイントレンド | Behance / Dribbble / Pinterest / Awwwards |
| 映像・コンテンツ | YouTube急上昇 / TikTokトレンド / Vimeo Staff Picks |
| デジタル商品 | Gumroad / Etsy(デジタル) / AppStore急上昇 |
| 海外ビジネス | Product Hunt / IndieHackers / Twitter(X)バズツイート |
| AIツール最前線 | Hugging Face / AI系Newsletter |

## リサーチ部の掟
- 「なんとなく面白い」で終わらせない。必ず「なぜバズっているか」を分析する
- 日本未上陸・日本語化されていないものを優先して発掘する
- 週1回、CEOに「週次トレンドレポート」を提出する
