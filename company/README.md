```

    ↕ ここだけで会話する
  🏢 CEO（00_CEO/）
```

---

## この会社の3つの原則

**1. オーナーとCEOだけが話す**
→ 複雑な情報はCEOがすべて整理してから届ける

**2. 人と会わなくていいビジネスモデル**
→ デジタル販売・非同期・テキストベースを最優先

**3. 体調良い日も悪い日も、少しずつ前に進む**
→ 1日5分でもOK。止まらないことが最強の戦略

---

## 今すぐやること（体調に合わせて選ぶ）

### 🟢 体調良い日（30分）
`00_CEO/owner_profile.md` を埋める
→ スキル・使えるツール・週の空き時間を記入

### 🟡 普通の日（10分）
「私はデザイン・映像ディレクターです。今日何をすべきですか？」
→ Claude Codeに聞くだけ。CEOが答えます

### 🔴 しんどい日（0分でOK）
何もしなくていい日も会社は存在し続ける
→ 回復することが今日の仕事
