# スキル：見積書作成

## いつ使うか
案件の概要が決まり、クライアントに金額を提示するとき。

## 手順
1. クライアント情報と案件詳細を確認
2. 下記テンプレートに記入
3. 「修正回数」「著作権帰属」を必ず明記
4. PDF化して送付

## 見積書テンプレート

```
件名：【お見積書】〇〇制作について

株式会社〇〇
〇〇様

お世話になっております。
以下の通りお見積もりをご提示いたします。

━━━━━━━━━━━━━━━━━
見積金額：金 〇〇〇,〇〇〇円（税込）
有効期限：発行日より30日
━━━━━━━━━━━━━━━━━

【内訳】
| 項目 | 数量 | 単価 | 金額 |
|---|---|---|---|
| 企画・構成費 | 1式 | 〇〇,〇〇〇円 | 〇〇,〇〇〇円 |
| 制作費 | 1式 | 〇〇,〇〇〇円 | 〇〇,〇〇〇円 |
| 修正対応（2回まで）| 込み | - | - |
| 小計 | | | 〇〇〇,〇〇〇円 |
| 消費税（10%） | | | 〇〇,〇〇〇円 |
| **合計** | | | **〇〇〇,〇〇〇円** |

【条件】
- 修正回数：2回まで（以降は別途お見積もり）
- 著作権：納品・入金完了後にクライアントへ帰属
- お支払い：納品後30日以内に銀行振込
- 納期：ご発注から〇週間

ご不明点はお気軽にご連絡ください。
```

## 注意点
- **修正回数は必ず明記**（無制限修正は赤字の元）
- 著作権の取り扱いを事前に決める（帰属 or ライセンス）
- 消費税：売上1000万円超で課税事業者になるため要確認
