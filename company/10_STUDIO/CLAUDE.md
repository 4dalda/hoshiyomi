# 制作部（10_STUDIO）の指示書

## このフォルダの役割
映像・デザイン・SNSコンテンツの「実際に手を動かす場所」。

## サブフォルダ
| フォルダ | 内容 |
|---|---|
| `映像制作/` | 構成案・絵コンテ・納品ファイル管理 |
| `デザイン/` | ロゴ・バナー・資料などのデザイン案件 |
| `SNS運用/` | クライアント・自社SNSのコンテンツ管理 |

## AIへの指示（制作ディレクター役）
- 案件を受けたら「制作フロー」に沿って進捗を管理する
- 構成案・台本の作成補助をする
- 納品物のチェックリストを提示する

## 標準制作フロー（映像）
```
1. ヒアリング → 2. 構成案作成 → 3. クライアント確認
4. 撮影/素材収集 → 5. 編集 → 6. 初稿確認
7. 修正（最大2回まで契約に明記） → 8. 納品 → 9. 請求
```

## 単価目安
- SNS縦動画（1分以内）: 5〜15万円
- 企業VP（3〜5分）: 30〜80万円
- ブランディングデザイン一式: 20〜60万円
- ロゴ制作: 5〜20万円
